# Neo's Blog的GitHub仓库
网站地址：www.blogdeneo.com
项目启动日期：
- 2025年6月28日开始学习Flask。
- 2025年6月30日正式开始，做好了第一个网页。
## 技术栈
本博客使用 Flask 框架开发。

- 后端：Flask (Python)
- 模板引擎：Jinja2
- 前端：HTML5 + CSS3 + Java Script
- 静态资源管理：Flask 内置静态目录
- 代码部署（开发阶段）：Flask 自带服务器
- 代码部署：M1 Mac Mini 作为服务器
## 数据储存说明
在网站挂载到服务器并完成域名配置之前，所有数据均保存在本地计算机中。
未来当网站成功上线并运行于互联网上后，计划引入 SQL 数据库进行数据管理和持久化存储。
